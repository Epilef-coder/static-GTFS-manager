The service_id is referred to when defining trips. Typically there are different schedules followed on working days and holidays.

The service_id's get used in TRIPS table to define which days of the week a trip operates on. This allows for different timings on weekends.

Use the Tools > Delete ID section to delete a calendar service. Note that because the service_id field is required in trips table, deleting a calendar service will need cascading deleting of all trips and timings data associated with it. It would be a better option to change the service_id instead.

If you have made any additions or changes that you wish to revert and you haven't pressed the big "Save" button below, you may simply reload this page, and all changes will be dropped. Conversely, make sure to Save to preserve any changes you make.
